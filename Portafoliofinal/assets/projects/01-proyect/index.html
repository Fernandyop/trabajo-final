Sobre el proyecto

Para este proyecto, se me encomendó el desarrollo de una página web funcional en WordPress, empleando el constructor visual Elementor para facilitar la personalización y la gestión del contenido. Además, se requería incluir una animación implementada con JavaScript, lo que añadió un componente interactivo y dinámico al diseño. El objetivo principal era combinar funcionalidad y estética, garantizando que la página no solo cumpliera con los requisitos técnicos, sino que también ofreciera una experiencia de usuario atractiva y profesional.
